\# Regras de Negócio



\## IPOC



O IPOC é composto por:



\- CNPJ raiz

\- modalidade

\- tipo do cliente

\- código do cliente

\- contrato



O sistema valida automaticamente divergências na composição.



\## Característica Especial



A característica especial 20 pode ser incluída automaticamente via planilha de apoio, preservando outras características existentes na operação.



\## Relacionamento do Cliente



O campo de início de relacionamento pode ser ajustado utilizando a menor data de contratação encontrada para o cliente.



\## Processamento XML



Os XMLs são processados em lote utilizando leitura incremental para suportar grandes volumes de dados.



\## Validação Regulatória



As rotinas foram desenvolvidas para apoiar validações pré-envio ao Banco Central.
